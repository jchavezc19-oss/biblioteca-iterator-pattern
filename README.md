### Justificación de Recorridos
- **Algoritmo Alfabético (Jorge):** Clona la colección base y ejecuta un ordenamiento estable en memoria $O(N \log N)$. Es ideal para búsquedas ordenadas por cara al usuario.
- **Algoritmo de Disponibilidad (Jairo):** Realiza saltos lógicos secuenciales sobre la misma colección sin usar memoria adicional $O(1)$ en espacio. Ideal para auditorías rápidas.
